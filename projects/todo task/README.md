# Smart Task Manager

A simple, modern web-based task manager built with Flask and Bootstrap.

## Features
- Add, edit, delete, and complete tasks
- Set deadlines and categories for tasks
- Search and sort tasks by deadline, category, or completion status
- Responsive and clean UI with Bootstrap
- Animated countdown for task deadlines

## Project Structure
```
app.py                # Main Flask application
requirements.txt      # Python dependencies
students.csv          # Sample CSV data (not used in app)
static/style.css      # Custom CSS styles
static/               # Static files (CSS, images)
templates/index.html  # Main page template
templates/edit.html   # Edit task template
Smart Task Manager.png# Screenshot of the app
```

## Getting Started

1. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```
2. **Run the app**
   ```bash
   python app.py
   ```
